## 图片优化建议

1. **格式**: 使用 JPG 格式，质量设置为 80-85%
2. **压缩**: 确保图片文件大小在 200KB 以内
3. **响应式**: 考虑提供不同尺寸的图片版本
4. **Alt文本**: 为每张图片添加合适的 alt 属性
